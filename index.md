# Welcome to HPC & LINUX Documentation

Welcome to our **HPC & LINUX** portal! This site is dedicated to helping you get started with **High Performance Computing**, **Linux Administration**, **Cluster Management**, **Lustre File Systems**, **GPUs**, and **InfiniBand**.

We provide extensive guides and resources to help you manage HPC clusters, set up Lustre storage, and work with GPUs.

## Documentation

Browse through our comprehensive documentation to learn about various topics, including:

- **High Performance Computing (HPC)**: A guide to setting up and managing large-scale computing resources.
- **Linux Administration**: Tutorials on Linux systems, commands, and best practices.
- **Cluster Management**: Instructions on how to manage and optimize high-performance computing clusters.
- **Lustre File System**: A detailed guide on deploying and maintaining Lustre for parallel file storage.
- **GPUs & InfiniBand**: Instructions on using GPUs for high-performance computing and optimizing performance with InfiniBand.

Start with our **[documentation](/documentation/)** to learn about setting up your environment and getting the most out of your HPC system.
