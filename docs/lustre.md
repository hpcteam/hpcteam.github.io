# Lustre File System

## Overview
Lustre is a high-performance parallel file system used in HPC.

## Core Components
- MGS – Management Server
- MDS – Metadata Server
- OSS – Object Storage Server
- OST – Object Storage Target

## L2 Tasks
- OST recovery
