# **Applications & Libraries**

This section lists the **commercial** and **open-source software**, **libraries, compilers, and tools**  
I have worked with in high-performance computing (HPC) environments to manage computational workloads. I also have experience **installing** both commercial and open-source applications in these environments, ensuring everything works smoothly.

### **Commercial Applications**

* **ANSYS Fluent** (v242) – Used for simulating fluid dynamics and other engineering problems.
* **ANSYS CFX** – CFD tool for high-performance simulations.
* **STAR-CCM+** – Multidisciplinary simulation software.
* **Altair FEKO** – Used for electromagnetic simulations and antenna design.
* **HEEDS** – Software for optimizing simulation-based designs.

### **Open-Source Applications**

* **OpenFOAM** (1912, 2112, 2312) – Used for simulating fluid dynamics.
* **SU2** (8.0.0, 8.1) – Open-source software for fluid flow and multi-physics simulations.
* **LAMMPS** – Molecular dynamics simulation software.
* **ParaView** (5.13.0) – Tool for visualizing and analyzing simulation results.
* **Scotch** (7.0, 7.0.10) – Used for graph partitioning and mesh decomposition.
* **METIS** (5.2, 5.2.1) – Tool for graph and mesh partitioning.

### **Core Libraries**

* **BLAS / LAPACK** (GCC 3.10.0, OpenBLAS 0.3.18) – Used for mathematical operations like linear algebra.
* **HDF5** (1.8.21, 1.12.1) – For storing and managing large datasets.
* **NetCDF** (4.1.0, 4.7.0) – For sharing and storing scientific data.
* **FFTW** (3.3.10) – For fast Fourier transforms (used in many scientific applications).
* **Boost** (1.77.0) – A set of C++ libraries for various system tasks.

### **Compilers & MPI Libraries**

* **GCC** (11.2.0, 14.2.0) – Standard compilers for building applications.
* **AOCC/AMD** (4.2.0, 5.0.0) – Optimized compilers for AMD hardware.
* **LLVM** (11.1.0, 13.0.0, 18.1.8) – Compiler for optimization and parallelism.
* **OpenMPI** (4.0.3, 5.0.7) – Open-source library for parallel computing.
* **MPICH** (3.4.2, 4.2.0) – MPI implementation for distributed computing.
* **CUDA Toolkit** (11.5 - 12.4) – For GPU-accelerated applications.
* **cuQuantum** (25.06) – A library for quantum computing on GPUs.

### **Python Environments & Libraries**

* **Python 3.9.18, 3.10.16, 3.11** – Used for scientific computing and data processing.
* **Custom Python Environments** – Tailored Python environments for specific tasks.
* **NumPy, SciPy, Pandas** – Libraries for numerical analysis and data manipulation.
* **Matplotlib, Seaborn** – Tools for creating graphs and visualizations.
